Placements

The placement session for any company generally has 3 â€“ 4 rounds. The first round is the written test, consisting of quantitative aptitude, reasoning (logical ability), English (verbal ability) and computer science subjects (C, C++, Data Structures, Algorithms, DBMS, OS, Networks, etc.). For some companies, you might also have a coding round, which is sometimes taken as a separate round and sometimes is clubbed with the written test. Further rounds are interview rounds, the first two usually being the technical round and successive rounds being the HR round.

Quantitative Aptitude

Numbers, Mensuration, Geometry, Algebra, Permutation & Combinations, Probability, Ratio Proportaion & variations

Reasoning (Logical Ability)

Missing Numbers, Odd One Out, Data Sufficiency, Assumptions and Conclusions, Courses of Action, Puzzles, Syllogism, Cubes

English (Verbal Ability

Passage/Sentence Rearrangementm, Error Detection and Correction, Fill in the Blanks, Comprehension Passages

Data Structures

Array, Linked List, Stack, Queue, Binary Tree, Binary Search Tree, Heap, Hashing, Graph, Matrix, Advanced Data Structure

Algorithms

Analysis of Algorithms, Searching and Sorting, Greedy Algorithms, Dynamic Programming, Pattern Searching, Backtracking, Divide and Conquer,

Operating Systems

Basics, System Structure, CPU Scheduling, Process Synchronization, Deadlock, Processes & Threads, Memory Management, File and Disk Management
